# Content Pipeline — Пайплайн контента

Активировать: когда нужно создать пост для Telegram-канала.

**Цель:** Превратить сырую идею в готовый пост, который прочитают.

---

## Зачем

80% Telegram-контента — мусор. Анонсные посты без структуры, "подписывайтесь на канал" в конце каждого сообщения. Хороший контент = структура + конкретика + действие.

---

## Этапы пайплайна

### 1. 💡 Идея
```python
def generate_idea(topic: str) -> str:
    """Сгенерировать идею поста на основе темы"""
    return f"""
    Тема: {topic}
    Формат: проблема → решение → результат
    Аудитория: начинающие разработчики в РФ
    Цель: дать конкретную пользу за 2 минуты чтения
    """
```

**Правила:**
- Одна идея = один пост
- Начинай с проблемы, не с решения
- Конкретика > абстракции

### 2. 📝 Структура
```
🪝 Хук (первые 2 строки — зацепят или потеряют)
📊 Проблема (2-3 предложения)
💡 Решение (конкретные шаги/код)
📈 Результат (что получилось)
🔗 CTA (призыв к действию)
```

**Пример:**
```
🪝 Твой AI-агент тратит $5 на задачу, которую можно сделать за $0.01. Вот как.

📊 Все используют GPT-4 для каждой мелочи. Парсинг JSON — GPT-4. Генерация текста — GPT-4. Даже суммирование — GPT-4. Это как ездить на танке в магазин.

💡 cost-aware-orchestrator автоматически выбирает дешёвую модель для простых задач:
- Groq Llama 3.1 8B → парсинг, суммирование (0.05$)
- DeepSeek V3 → анализ данных (0.14$)
- GPT-4 → только сложные задачи (3.00$)

📈 Результат: 95% экономии на API-вызовах. Задача за 0.01$ вместо 5$.

💬 Какую модель вы используете для простых задач? Напишите в комментариях 👇
```

### 3. ✏️ Редактура
- Убери воду и повторы
- Замени абстракции на примеры
- Добавь конкретные цифры
- Проверь длину (2000-4000 символов оптимально)

### 4. 📤 Публикация
```bash
# Через бота-привратника
python posting_bot.py --channel @channel_name --post post.md

# Или через скрипт
python publish_posts.py --channel @channel_name
```

## SEO для Telegram

| Элемент | Совет |
|---------|-------|
| Заголовок | Эмодзи + суть за 3 слова |
| Первые 2 строки | Хук —决定了 читатель остается или листает |
| Код | В ```` ``` ```` — копируется на телефоне |
| Скриншоты | Визуал > текст для мобильных |
| CTA | "Подпишись" в середине лучше, чем в конце |

---

## Правила

1. **ВСЕГДА** начинай с проблемы, не с решения
2. **ВСЕГДА** добавляй конкретные цифры и примеры
3. **ВСЕГДА** заканчивай CTA (вопрос, просьба, действие)
4. **НИКОГДА** не пиши "подписывайтесь на канал" в конце — это убивает вовлечённость
5. **НИКОГДА** не превышай 4000 символов — люди не дочитают
